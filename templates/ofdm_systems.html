OFDM Formulas
Key Calculations:

Symbol Duration:
Ts = 1 / Δf
Ttotal = Ts × (1 + CP)

Bits per Symbol:
b = log₂(M)
where M = modulation order

RE Data Rate:
RRE = (b × Rcode) / Ttotal

RB Data Rate:
RRB = RRE × Nsc × Nsym

Spectral Efficiency:
η = Rtotal / Btotal (bits/s/Hz)

• 15 kHz subcarrier spacing
